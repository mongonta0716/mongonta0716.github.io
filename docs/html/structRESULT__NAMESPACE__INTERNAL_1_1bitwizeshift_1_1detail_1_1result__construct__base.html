Detailed Description

template<typename T, typename E>
struct RESULT_NAMESPACE_INTERNAL::bitwizeshift::detail::result_construct_base< T, E >

Base class of assignment to enable construction and assignment.

This class is used with several pieces of construction to ensure trivial constructibility and assignability:

  • result_trivial_copy_ctor_base
  • result_trivial_move_ctor_base
  • result_copy_assign_base
  • result_move_assign_base
Template Parameters
Tthe value type
Ethe error type
