Allocate bus for screen communication.

Parameters
transactionIf true, transaction processing is performed.
Note
Although bus allocation and release are automatically performed when drawing functions are called, using startWrite and endWrite before and after the drawing process suppresses bus allocation and release and improves drawing speed. In the case of electronic paper (EPD), drawing after startWrite() is reflected on the screen by calling endWrite().
